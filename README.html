1. What is Gate Mail Pro?

Gate Mail Pro is an elegant, fast, and feature-rich desktop application designed to generate, manage, and monitor disposable temporary email addresses. It acts as a centralized gateway, providing seamless access to several temporary email servers without requiring you to visit clunky websites, endure advertisements, or face invasive tracking scripts.

Whether you need a quick email to verify a forum account, download a trial, or protect your identity online, Gate Mail Pro puts you in complete control — right from your desktop.

2. Understanding Disposable Email

A disposable (temporary) email address is a real inbox created on the fly for a specific purpose. It operates independently of your private or personal identity, allowing you to interact with websites and services without risking your data.

Key Advantages:

  • Eliminate Spam: Register on forums, download e-books, or access quick trials without clogging your primary inbox with advertisements and promotional newsletters.
  • Preserve Privacy: Prevent data breaches from exposing your real passwords and email address, keeping your digital footprint secure.
  • Instant Trialing: Sign up and test software platforms instantly without filling out tedious personal questionnaires.

3. Download & Installation

Gate Mail Pro is distributed in two formats. Choose the one that best suits your needs:

📦 Installer Version 💾 Portable Version (ZIP)

Installer Version

A traditional setup wizard that installs the application on your system with Start Menu shortcuts. Simply run the downloaded .exe file and follow the on‑screen instructions.

Portable Version

A ZIP archive containing the application files — no installation required. Simply extract the archive to any folder on your computer (or a USB drive) and run the executable directly. Alongside it you will find several important folders: languages, Changes, and sounds. Keep these folders in the same directory — the program reads them at startup.

First Launch

On first launch you will be asked to select your preferred interface language. The main window will then open, and you are ready to create your first temporary email account.

4. Interface Overview & Screen Reader Navigation

Gate Mail Pro was built with accessibility in mind. Every button, list, text field, and menu item is properly labeled and exposed to assistive technology such as screen readers (NVDA, JAWS, Narrator). The interface is divided into four main screens that you switch between using the navigation controls or keyboard shortcuts.

Screen 1 — Your Saved Accounts Accessible

This is the home screen. It displays a list of all temporary accounts you have created and saved. Each entry shows the email address and the date it was created. Below the list you will find four buttons:

  • Login (Enter): Opens the inbox for the selected account.
  • Delete Selected (Delete): Removes only the highlighted account.
  • Delete All Accounts: Wipes every saved account after a confirmation prompt.
  • Create New Account: Moves you to the account creation screen.

Screen Reader Tip: Use the Tab key to move through the list and buttons. Press Enter on a highlighted account to open it. Press Delete to remove the selected account without touching the mouse.

Screen 2 — Create New Account Accessible

Here you select a domain from the dropdown, type a custom username or click Generate Random, and confirm creation. Some domains generate the username automatically for you — in that case the username field will be hidden and a message will inform you.

Screen Reader Tip: Focus lands on the domain dropdown. Press Tab to move through the controls. Type your desired name or press the Random button. Press Enter on Confirm & Create to finish.

Screen 3 — Inbox Dashboard Accessible

Once you have created or logged into an account, you land here. The top section shows your ready‑to‑use email address with a Copy Email button. Below that, a refresh button, an auto‑refresh checkbox, a search field, a sort dropdown, inbox statistics, and the message list. Double‑clicking a message opens it in a dedicated reading window.

Screen Reader Tip: The email address is in a read‑only text field — your screen reader will announce it as soon as the dashboard opens. Use Tab to reach the message list and arrow keys to browse subjects. Press Enter to open a message.

Screen 4 — Message Reader Accessible

When you open a message, you land on a dedicated reading screen. The top bar has a Back button and an Actions Menu button. The message content is displayed in a list format with the sender, subject, and body clearly separated. You can right‑click or press the Applications key to access the context menu with copy, extract, and save options.

Screen Reader Tip: The message content is announced line by line. Use arrow keys to navigate. Press Applications or Shift+F10 to open the context menu for additional actions.

6. Creating & Managing Accounts

How to Create an Account

  1. From the main screen, click Create New Account or press Tab until you reach it and press Enter.
  2. Choose a domain from the dropdown.
  3. If the domain allows custom usernames, type your preferred name or click Generate Random. If the domain does not allow custom names, a message will inform you that the username is generated automatically.
  4. Click Confirm & Create. A loading dialog will appear while the account is being registered.
  5. Once created, the application automatically saves the account and opens the inbox dashboard.

Managing Saved Accounts

Every account you create is stored locally on your computer. They persist between sessions — you can close the program, reopen it later, and find all your accounts waiting on the main screen. From there you can:

  • Log into any account to check for new messages.
  • Delete individual accounts you no longer need.
  • Delete all accounts at once via the dedicated button (a confirmation dialog will appear).

Account Limit

New in V2.1.3: Gate Mail Pro allows a maximum of 2 accounts to be saved at any time. This limit helps maintain optimal performance and ensures reliable operation. If you need more accounts, you can delete unused ones and create new ones as needed.

Clearing All Application Data

If you wish to completely reset Gate Mail Pro to its factory state — removing all accounts, settings, language preferences, and sound selections — go to Settings > Clear All Data. The program will close automatically after the operation. Use this with caution, as the action cannot be undone.

7. Inbox Dashboard & Reading Messages

Viewing Your Email Address

At the top of the dashboard, your temporary email address is displayed in a read‑only text field. You can click the Copy Email button (or focus it with Tab and press Enter) to copy the address to your clipboard. Paste it wherever you need to receive a verification or confirmation message.

Checking for Messages

Use the Manual Refresh (F5) button to instantly check for new messages. If you are waiting for a time‑sensitive email (such as a verification code), enable the Auto Refresh checkbox. The program will automatically poll the server at your chosen interval and notify you when new messages arrive.

Configurable Auto‑Refresh Interval NEW

New in V2.1.3: You can now choose your preferred auto‑refresh interval from the Settings > Auto Refresh menu. Available options:

  • 10 seconds — For time‑sensitive verification codes.
  • 30 seconds — Default, balanced interval.
  • 1 minute — Moderate polling rate.
  • 5 minutes — Low‑frequency checking.
  • 10 minutes — Minimal server requests.

The selected interval applies immediately — no restart required. Your choice is saved between sessions.

Reading a Message

Double‑click any message in the inbox list (or highlight it with arrow keys and press Enter) to open it in the message viewer. The viewer shows the subject, sender, and full body content. From the viewer you can:

  • Copy the entire message to clipboard.
  • Copy the currently selected line.
  • Copy only the sender's name.
  • Open the message in Notepad for easier reading.
  • Save the message as an HTML file.
  • Extract links and numbers (see next section).

8. Extracting Links & Verification Codes

One of the most powerful features of Gate Mail Pro is the Smart Content Extraction tool. When you open a message, use the Actions Menu button or right‑click anywhere in the message area (or press the Applications key / Shift+F10) to open the context menu. You will see:

  • Copy Selected Line: Copies only the line currently highlighted in the message.
  • Copy Message: Copies the entire message body.
  • Open in Notepad: Launches Notepad with the full text for comfortable reading.
  • Save as HTML: Saves the complete message as a formatted HTML file to your chosen location.
  • Copy Sender Name: Copies only the sender's email address or name.
  • Extract Links & Numbers: A submenu that lists every URL and every number with 5 or more digits found in the message. Click any link to open it directly in your browser, or click any number to copy it to your clipboard. This is extremely useful for grabbing verification codes without manually searching through walls of text.
  • Copy All Extracted: Copies all found links and numbers at once.
  • Open Extracted Items in Notepad: Opens a new Notepad window containing all extracted links and numbers, one per line.

Screen Reader Tip: After opening a message, press Applications or Shift+F10 to open the context menu. Arrow down to "Extract Links & Numbers" and press Right Arrow to expand the submenu. All found links and numbers will be listed — press Enter on any of them to copy or open it.

9. Inbox Features: Search, Sort & Statistics NEW

New in V2.1.3: The inbox dashboard now includes powerful tools to help you manage your messages efficiently.

Inbox Search

A search field is located below the refresh controls. Type any text to instantly filter messages by sender name or subject. The filter is case‑insensitive and updates in real time as you type. When no messages match your search, a "No results found" message is displayed.

Sort Options

A sort dropdown is located next to the search field. You can choose between:

  • Oldest First: Messages are displayed in chronological order (oldest at the top).
  • Newest First: Messages are displayed in reverse chronological order (newest at the top).

Changing the sort order immediately updates the message list. The selected sort preference is remembered between sessions.

Inbox Statistics

Below the message list, a statistics line displays the total message count in the current inbox and the number of unique senders.

Quick Preview

Hover over any message in the inbox list to see a tooltip with the sender and subject information.

Export Emails NEW

You can export all emails in your inbox as individual HTML files. Go to the inbox dashboard and use the export function to save all messages to a folder of your choice. Each email is saved as a separate, well‑formatted HTML file with sender, subject, and body content preserved.

10. Menu Settings Explained

The menu bar is located at the top of the window and can be accessed by pressing Alt or clicking on it. The menus are organized into two main groups: Settings and Help.

Settings Menu

  • Language: Choose your preferred interface language. Gate Mail Pro automatically discovers language files placed in the languages folder. Supported languages include English and Arabic by default, with the ability to add more via custom translation files.
  • Theme: Toggle between Dark Mode (default) and Light Mode. The change takes effect immediately without restarting.
  • Font Size: Five preset levels are available — from Small (12px) to Maximum (20px). The default is Medium (14px). This setting helps users with low vision find their most comfortable reading size.
  • Inside App Sounds: Choose which sound plays when a new message arrives while the window is focused. Hover over a file name to preview it.
  • Outside App Sounds: Choose which sound plays when a new message arrives while the window is minimized or in the background.
  • Auto Refresh Interval: Choose how often the inbox is automatically checked — 10s, 30s, 1m, 5m, or 10m.
  • Clear All Data: Wipes all accounts, settings, and preferences. The application will close after confirmation.
  • Exit Program: Opens the close dialog where you can choose to exit completely or keep running in the background.

Help Menu

  • About App: Opens the README file in your web browser.
  • About Developer: Shows contact information for the developer and contributor.
  • Check for Updates: Connects to the update server and informs you if a newer version is available.
  • What's New: Displays the change log for the current version. You can opt to never see this window again by checking the checkbox at the bottom.

11. System Tray & Background Operation

Gate Mail Pro supports running silently in the background. When you close the main window (by clicking the X button or pressing Alt+F4), a dialog appears with three options:

  • Cancel: Return to the application without closing it.
  • Keep Running in Background: The window hides, but the application continues running. You will see its icon in the system tray (near the clock). The app will continue to check for new messages if auto‑refresh is enabled, and will show desktop notifications for incoming mail.
  • Exit Program: Completely terminates Gate Mail Pro and all its background processes.

Restoring from System Tray: Click the tray icon once to restore or hide the window. Right‑click the tray icon for a menu with Restore and Exit options.

Screen Reader Tip: Navigate to the system tray with Windows+B, arrow to the Gate Mail Pro icon, and press Enter to restore the window or Applications to open the tray menu.

12. Notification Sounds

The application includes two separate sound channels for incoming message alerts:

  • Inside App Sound: Played when the Gate Mail Pro window is active and visible.
  • Outside App Sound: Played when the application is minimized to tray or hidden.

You can change these sounds at any time via Settings > Inside/Outside App Sounds. Hover over any file name to hear a preview. Custom sound files can be added by placing them in the sounds folder next to the executable. The supported format is .wav.

13. Checking for Updates

To check if a newer version of Gate Mail Pro is available, go to Help > Check for Updates. The application will contact the update server and compare versions. If a newer version is found, a dialog will present you with two download options:

  • Installer Version: A traditional setup wizard that installs the application on your system.
  • Portable Version: A ZIP archive containing the portable executable — ideal for USB drives.

You can copy the download link or open it directly in your browser from this dialog.

Automatic Check: Gate Mail Pro quietly checks for updates a few seconds after launching. If an update is found, the same dialog will appear automatically.

14. Unique Features Summary

Unlike basic web‑based disposable mail services that delete your inbox the moment you refresh or close your browser, Gate Mail Pro offers key desktop utilities:

  • Multi‑Server Routing: Choose from various temporary email servers depending on your needs.
  • Account Persistence: Your generated accounts are saved locally. You can close the program, open it tomorrow, and log back into your temporary inbox to read new verification messages.
  • Smart Content Extraction: Right‑click on any incoming message to extract links and verification numbers instantly. This eliminates the need to scan through long, HTML‑heavy emails manually.
  • Background Operation: Minimize the application to your system tray. It will run silently in the background, showing desktop alerts only when new mail is detected.
  • Complete UI Personalization: Customize font sizes, toggle between high‑quality light and dark modes, and easily load custom translation files.
  • Full Keyboard & Screen Reader Support: Every function is accessible via keyboard shortcuts, and all UI elements are properly labeled for assistive technology.
  • Configurable Auto‑Refresh: Choose your preferred polling interval from 10 seconds to 10 minutes.
  • Inbox Search & Sort: Quickly find messages by sender or subject, and sort by oldest or newest first.
  • Email Export: Export all inbox messages as individually formatted HTML files.
  • Smart Token Refresh: The application automatically refreshes authentication tokens to maintain your session across extended periods.

15. Troubleshooting & FAQ

Q: The program shows "Network error or service unavailable."

This means the temporary email server could not be reached. Check your internet connection. Some servers may be temporarily down. Try selecting a different domain from the creation screen — Gate Mail Pro supports multiple independent servers.

Q: I created an account but no messages appear in the inbox.

Press F5 or click Manual Refresh. If you are expecting a verification email, enable Auto Refresh and wait up to 60 seconds. Also ensure you have given the correct email address to the website or service you are registering with — use the Copy Email button to avoid typos.

Q: Sounds are not playing after I compiled the program.

Verify that the sounds folder exists in the same directory as the executable and contains valid .wav files. The application looks for sounds relative to its executable location.

Q: The interface language did not change after I selected a new language.

After changing the language, the entire interface rebuilds itself. This takes effect immediately. If you see mixed languages, ensure the language file inside the languages folder is complete and correctly formatted.

Q: How do I add a new language?

Create a .lang file (JSON format) inside the languages folder. The application will detect it automatically on the next launch. You can use the existing English or Arabic files as templates.

Q: Can I use this application on other operating systems?

Gate Mail Pro is currently designed and tested for Windows. However, since it is built with cross‑platform technologies, advanced users may be able to run the source code on other operating systems with minor adjustments. Pre‑compiled executables are only provided for Windows at this time.

Q: Is my data sent anywhere?

All account data is stored locally on your computer. No personal information or email content is transmitted to any server other than the temporary email servers themselves (which is necessary to fetch your messages). An anonymous ping is sent on startup for usage statistics, containing no identifiable information.

Q: Why is there a limit of 2 accounts?

The account limit of 2 helps maintain optimal performance and ensures reliable operation across all supported servers. If you need more accounts, you can delete unused ones and create new ones as needed.
